เหตุผลที่เลือกราชมงคลล้านนา

ในช่วงที่ผมกำลังจะจบปวช. แม่ผมที่เคยเป็นศิษย์เก่า แนะนำให้ผมไปงานเปิดบ้านของ "มหาวิทยาลัยราชมงคลล้านนา เชียงใหม่" ผมจึงได้ลองไปดู และได้เดินสำรวจแต่ละคณะจนผมได้เจอ คณะวิศวะคอมพิวเตอร์ ที่นำเสนอกิจกรรมเกี่ยวกับหุ่นยนต์ ทำให้ผมเริ่มสนใจที่จะเขียนระบบให้กับAI เป็น อย่างมาก ทำให้ผมเชื่อว่ามหาวิทยาลัยแห่งนี้จะสามารถนำความรู้มาให้แก่ผม ผมจึงเลือกมหาลัยแห่งนี้เพื่อขัด เกลาความรู้ความสามารถของตนเอง ผมจึงเลือกที่นี้เป็น "มหาลัยที่ดีที่สุดสำหรับผม"

เหตุผลที่เรียนวิศวะคอมพิวเตอร์

ผมเป็นชื่นชอบคอมพิวเตอร์ตั้งแต่วัยเด็ก ผมจึงศึกษาเกี่่ยวกับคอมพิวเตอร์มาหลายด้าน ทั้งด้านการ ตัดต่อรูปภาพ วีดีโอ ออกแบบสื่อ เครือข่าย อุปกรณ์คอมพิวเตอร์ พอโตมาช่วงม.ต้น ผมได้ทำระบบเซิฟเวอร์ เกมMinecraft และได้เริ่มศึกษาการเขียน โปรแกรมภาษาjava และ C# ตามด้วย HTML CSS ฯลฯ ทุกครั้งที่แก้bugได้สำเร็จผมมักที่จะดีใจไปกับมัน ซึ่งทำให้ผมชื่นชอบและหลงไหลในการเขียนโปรแกรม อย่างมากซึ่งในยุคดิจิทอลแบบนี้ ก็ใช้เวลาไม่นานที่ทำลายความสงสัยของผมเกี่ยวกับคอมพิวเตอร์และ การเขียนโปรแกรม ซึ่งเป็นสิ่งที่น่าค้นหาและให้ความสนใจมากที่สุดจนถึงปัจจุบัน
